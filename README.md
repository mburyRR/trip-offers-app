# React trip offers test app (PWA)

## React + Vite + Context API + Infinite Scroll

---------

### Instal dependencies
```
yarn
```

### Run development website locally
```
yarn dev
```

### Build
```
yarn build
```

* Build files will be placed in the "/dist" folder by default.


### Run production build website locally
```
yarn preview
```
